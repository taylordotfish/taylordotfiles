Dependencies
============

taylordotfiles requires the following dependencies. Here, Debian package names
are used.

Base:

* awk
* gcc
* jq
* m4
* make
* lua5.3
* python3

Graphical environment:

* brightnessctl (if laptop)
* dispwin
* gnome-icon-theme
* hsetroot
* i3
* libx11-dev
* libxinerama-dev
* libxrandr-dev
* logrotate
* lxpolkit
* xsettingsd

Optional:

* cmark or pulldown-cmark
* libpulse-dev
* rsync
